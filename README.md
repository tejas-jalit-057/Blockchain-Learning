## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/tejas-jalit-057/Blockchain-Learning.git
cd Blockchain-Learning

Install dependencies (if applicable):

npm install

or

pip install -r requirements.txt
▶️ Usage

Run examples or scripts:

node filename.js

or

python filename.py

Explore different folders to understand each concept step-by-step.
